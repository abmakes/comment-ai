# A T3app that uses open AI to assist teachers

While working as and ESL teacher a lot of my time was being taken up by writing comments for students. As an exercise in working with tailwind CSS and the openAi api I decided to build this app. It has helped other teacher's and speed up work and allows me more time for coding.

The setup indcluded using T3app, typescirpt and Tailwind:
- Designed in Figma 
- Styled using Tailwind CSS.
- Clerk for user and authentication management 
- GPT3.5 to generate commetns based on the users input.

View live app: [Comments-AI](https://comment-ai.vercel.app/)
